# Complex-Network-Analysis-Project
End-to-end graph analysis of a HEP-PH dataset: from degree distribution (CCDF) and assortativity to community detection and structural resilience.  Developed during the Networking course for the First Level Professional Master in Cybersecurity (2024/25, Università di Pisa).
Inside /code you can find the original dataset in .ncol extension and a little Python script to conver the .ncol in .txt. The full project was developed in Jupyter.
